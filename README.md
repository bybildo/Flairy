# Flairy Version 0.5

## Опис версії  

### Створено:  
- **Сторінки керування для бізнес акаунту**  

### Партнер має такі можливості:
- **Переглянути профіль** _не реалізовано_ 
- **Додавати та переглядати літаки** 
- **Додавати та переглядати маршрути**
- **Додавати та переглядати рейси**

### Додатково:
**🔹Усі дані введені партнером зберігаються у нашу локальну базу даних** 

**🔹 Робота з базою даних**  
- Створення запитів вибірки інформації та запитів додавання до таблиці:  
  - Літаків
  - Маршрутів 
  - Рейсів
 
**🔹 Функціонал додавання літаків**  
- Користувач може вільно редагувати салон літака

## Коментарі  
Основний фокус у цій версії – створення функціоналу для партнерів
