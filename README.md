# Flairy (Flight Ticket Sales System)

**Статус проєкту: Завершено (Курсова робота)**

Десктопний застосунок для автоматизації процесу бронювання та продажу авіаквитків. Проєкт реалізований як курсова робота, що демонструє інтеграцію графічного інтерфейсу користувача з реляційною базою даних.

## Технологічний стек
* **Мова програмування:** C# (.NET)
* **Графічний інтерфейс:** WPF (Windows Presentation Foundation)
* **База даних:** MS SQL Server

## Основний функціонал
* **Керування рейсами:** Перегляд актуального розкладу літаків у реальному часі.
* **Пошук та фільтрація:** Пошук квитків за напрямком, датою вильоту та іншими параметрами.
* **Бронювання:** Система оформлення квитків із прив'язкою до персональних даних пасажира.
* **Валідація:** Автоматична перевірка коректності введення даних та контролю наявності вільних місць.
